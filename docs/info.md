## How it works

Telecommunications uses a standard called DS0. It runs at 64kbps and is a method of interpreting voice frequency traffic. I work at a manufacturer where we produce DS0 to IP converters, as well as many other serial methods to IP. Because of my experience there I wanted to create a tiny-tapeout program that uses a standard we work with. When we program boards to work with DS0 we use a CPLD and not a ASIC because they are more versatile, but the idea still works for an ASIC if I write the verilog.
